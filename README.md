# Habu Gallery

撮影した写真を「良い感じ」に展示するフォトギャラリーサイトです。
メーソンリー（高さ可変のグリッド）レイアウト＋ライトボックス（クリックで拡大）で構成しています。

公開URL: https://kanami-habu1122.github.io/gallery/

## 写真の登録方法（常設で公開する写真）

1. 写真ファイルを [`photos/`](photos/) フォルダに入れる（`.jpg` / `.png` / `.webp` など）
2. [`photos.js`](photos.js) のリストに1行追加する:

   ```js
   { src: "photos/my-photo.jpg", title: "タイトル", tag: "live" },
   ```

   - `src` … `photos/` からのパス（必須）
   - `title` … 写真のタイトル（任意・ライトボックスで表示）
   - `tag` … カテゴリ（任意・上部フィルターに使われる 例: `live` / `portrait` / `stage`）

3. コミットして push すると、数分で公開サイトに反映されます。

> 最初から入っている `photos/sample-01.svg` 〜 `sample-06.svg` はプレースホルダです。
> あなたの写真に差し替えるか、不要なら削除して `photos.js` からも消してください。

## 「写真を並べてみる」機能について

サイト下部のドロップゾーンに写真をドラッグすると、その場でギャラリーに並べて見た目を試せます。
ただしこれは**閲覧者のブラウザ内だけ**に保存される機能で、サーバーには送られず、他の人には見えません。
（GitHub Pages は静的サイトのため、アップロードの永続保存はできません。常設公開は上記の `photos.js` で行います。）

## ローカルで確認する

`index.html` をブラウザで開くだけで動作します（写真リストは `photos.js` から読み込むため、ローカルでもそのまま表示されます）。

## 構成

| ファイル | 役割 |
|---|---|
| `index.html` | ギャラリー本体（レイアウト・ライトボックス・追加機能） |
| `photos.js` | 展示する写真のリスト |
| `photos/` | 画像ファイル置き場 |
